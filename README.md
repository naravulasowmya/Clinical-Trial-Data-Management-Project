Clinical Trial Data Management Project

Overview

This project demonstrates an end-to-end Clinical Trial Data Management workflow using Microsoft Excel. It includes participant demographics, medical history, screening, study visits, laboratory results, vital signs, adverse events, query management, protocol deviations, and a dashboard.

Tools Used

- Microsoft Excel
- Excel Formulas
- Charts & Dashboard
